Conway's Game of Life 是一个在 2D 方形网格上玩的元胞自动机。网格上的每个方块（或“单元格”）可以是活的，也可以是死的，它们根据以下规则演化：

  • 任何活细胞少于两个活的邻居都会死亡（称为人口不足）。
  • 任何具有三个以上活邻居的活细胞都会死亡（称为人口过剩）。
  • 任何有两三个活邻居的活细胞都会不变地活到下一代。
  • 任何只有三个活邻居的死细胞都会复活。

细胞的初始构型可以由人类创造，但此后的所有世代都完全由上述规则决定。游戏的目标是找到以有趣方式进化的模式——这是人们现在已经做了 50 多年的事情。

首先了解游戏的最佳方式是通过上面的 LifeViewer 或下载生活模拟程序 Golly 来玩它。绘制一些图案，然后按“播放”按钮查看它们是如何演变的。一旦你熟悉了这个游戏，你可以看看LifeWiki或Catagolue来浏览关于它的知识，或者你可以阅读这本书，以获得更直接的生活世界的指南。

大多数新的生命发现都会在 ConwayLife.com 论坛和 Conwaylife Lounge Discord 服务器上公布和讨论。
